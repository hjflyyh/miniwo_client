# Miniwo

Miniwo is an AI theatre sandbox where users can create virtual worlds, place AI NPCs inside them, and watch characters live, interact, and unfold stories in real time.

## Marketing Information

Miniwo lets users build imaginative virtual environments, create AI characters, and observe emergent stories generated through multi-agent interactions. The product combines sandbox creation, simulation gameplay, and AI-driven storytelling.

Miniwo is currently available through TestFlight for beta testing.

## Privacy Policy

Miniwo respects user privacy.

During TestFlight beta testing, we may collect limited information necessary to operate, improve, and test the app, including:

- Account or sign-in information
- App usage and interaction data
- Device and crash diagnostic information
- Feedback submitted by beta users

We do not sell user personal information.

Collected data is used only for app testing, product improvement, bug fixing, and user support.
